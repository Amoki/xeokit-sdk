    public generateCameraDataTexture(gl: WebGL2RenderingContext, camera: Camera, scene: Scene, origin: null | number[3]): BindableDataTexture source

    Generate and return a camera data texture.

    The texture will automatically update its contents before each render when the camera matrix is dirty, and to do so will use the following events:

    • scene.rendering event will be used to know that the camera texture should be updated
    • camera.matrix event will be used to know that the camera matices changed

    public generateTextureForColorsAndFlags(gl: WebGL2RenderingContext, colors: ArrayLike<ArrayLike<int>>, pickColors: ArrayLike<ArrayLike<int>>, vertexBases: ArrayLike<int>, indexBaseOffsets: ArrayLike<int>, edgeIndexBaseOffsets: ArrayLike<int>, solid: ArrayLike<boolean>): BindableDataTexture source

    This will generate an RGBA texture for:

    • colors
    • pickColors
    • flags
    • flags2
    • vertex bases
    • vertex base offsets

    The texture will have:

    • 4 RGBA columns per row: for each object (pick) color and flags(2)
    • N rows where N is the number of objects

    Params:

    NameTypeAttributeDescription
    gl WebGL2RenderingContext
    colors ArrayLike<ArrayLike<int>>

    Array of colors for all objects in the layer
    pickColors ArrayLike<ArrayLike<int>>

    Array of pickColors for all objects in the layer
    vertexBases ArrayLike<int>

    Array of position-index-bases foteh all objects in the layer
    indexBaseOffsets ArrayLike<int>

    For triangles: array of offests between the (gl_VertexID / 3) and the position where the indices start in the texture layer
    edgeIndexBaseOffsets ArrayLike<int>

    For edges: Array of offests between the (gl_VertexID / 2) and the position where the edge indices start in the texture layer
    solid ArrayLike<boolean>

    Array is-solid flag for all objects in the layer

    Return:

    BindableDataTexture

    public generateTextureForPositions(gl: WebGL2RenderingContext, positions: ArrayLike<int>): BindableDataTexture source

    Params:

    NameTypeAttributeDescription
    gl WebGL2RenderingContext
    positions ArrayLike<int>

    Array of (uniquified) quantized positions in the layer

    This will generate a texture for positions in the layer.

    The texture will have:

    • 1024 columns, where each pixel will be a 16-bit-per-component RGB texture, corresponding to the XYZ of the position
    • a number of rows R where R*1024 is just >= than the number of vertices (positions / 3)

    Return:

    BindableDataTexture

    public generateTextureForPositionsDecodeMatrices(gl: WebGL2RenderingContext, positionDecodeMatrices: ArrayLike<Matrix4x4>, instanceMatrices: ArrayLike<Matrix4x4>): BindableDataTexture source

    This will generate a texture for all positions decode matrices in the layer.

    The texture will have:

    • 4 RGBA columns per row (each column will contain 4 packed half-float (16 bits) components). Thus, each row will contain 16 packed half-floats corresponding to a complete positions decode matrix)
    • N rows where N is the number of objects

    Params:

    NameTypeAttributeDescription
    gl WebGL2RenderingContext
    positionDecodeMatrices ArrayLike<Matrix4x4>

    Array of positions decode matrices for all objects in the layer
    instanceMatrices ArrayLike<Matrix4x4>

    Array of geometry instancing matrices for all objects in the layer. Null if the objects are not instanced.

    Return:
